# Recreating the "If Wishes Could Kill [기리고]" App in Flutter

## Inspiration
While watching the series, seeing an app being developed during the episodes made me want to code something.  

At one point, part of the source code was briefly shown on screen, and I noticed it appeared to be built with Jetpack Compose. Because of that, I decided to recreate the app using Flutter.

I also missed coding without relying on AI, so I decided to not use it during this project

## How I Made It
- While searching online for the main GIF used in the app, I couldn't find it anywhere. Then I thought about checking the Play Store to see if the app actually existed — and luckily, it did.
- I downloaded the app and used a few tools to perform reverse engineering and gain access to the original assets.
- After recovering the assets, the rest of the process was recreating the interface and behavior in Flutter.
